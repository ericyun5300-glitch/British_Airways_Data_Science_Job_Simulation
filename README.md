# British Airways Job Simulation

This repository contains my professional solutions for the British Airways Job Simulation hosted on Forage. The project demonstrates end-to-end data processing, quantitative analysis, and predictive machine learning.

---

## 📂 Project Structure

### ✈️ [Task 1: Operational Lounge Analysis](Task_1_Operational_Analysis/)
* **Focus:** Operational data analysis & customer segmentation.
* **Key Tasks:** Regional grouping (Europe/NA/Asia&ME), capacity calculations, and lounge eligibility assessment based on loyalty tiers.

### 🤖 [Task 2: Predictive Modeling](Task_2_Predictive_Modeling/)
* **Focus:** Predictive machine learning pipeline.
* **Key Tasks:** Data preprocessing, feature engineering (`purchase_lead`, `flight_hour`), and RandomForest modeling.

---

## 🛠️ Tech Stack
* **Analytics:** Python (Pandas), Excel
* **ML:** Scikit-learn (RandomForestClassifier)

## 📊 Summary of Results
* **Booking Prediction Accuracy:** 73% (Verified via 5-Fold Cross-Validation)
* **Top Drivers:** 
  1. `purchase_lead` (14.6%)
  2. `flight_hour` (12.1%)
  3. `length_of_stay` (11.0%)
